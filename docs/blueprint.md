# **App Name**: Drashti's Digital Canvas

## Core Features:

- Personalized Introduction: A dynamic, engaging introduction highlighting Drashti's skills and experience, tailored to capture the attention of potential employers.
- Project Showcase: Visually appealing display of Drashti's key projects with descriptions and links to live demos or GitHub repositories. This will make use of the links provided: https://github.com/Drashti1605, https://www.linkedin.com/in/drashti-khanpara-3a3b83253/.
- Skills Matrix: Interactive matrix displaying Drashti's skills, with proficiency levels. This allows potential employers to quickly identify key competencies.
- Contact Form: A simple form that sends to khanparadrashti1605@gmail.com, allowing visitors to quickly reach out to Drashti.
- AI-Powered Style Guide Review: An AI tool that analyzes a design file (e.g., Figma) and provides feedback on adherence to established style guidelines, helping maintain consistency and brand integrity.  The reasoning "tool" will review aspects such as color palettes, typography, spacing, and component usage. 

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to reflect creativity and sophistication.
- Background color: Very light purple (#F3E5F5), almost white, to allow the work to pop.
- Accent color: Blue-purple (#3F51B5) to create contrast and highlight key information.
- Body font: 'Inter', a sans-serif font, to provide a modern and clean look for readability.
- Headline font: 'Space Grotesk', a sans-serif font, for a bold and modern title.
- Clean, minimalist icons that complement the overall design. These will represent skills, tools, and experience.
- A clean and structured layout with clear sections for easy navigation, emphasizing a user-friendly experience.
