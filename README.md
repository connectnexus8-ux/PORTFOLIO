# My Portfolio

This is the source code for my personal portfolio website, built with Next.js, Tailwind CSS, and ShadCN UI.

## Features

- Interactive project showcase with both a visual grid and a terminal mode
- Detailed sections for experience, skills, and achievements
- Smooth animations and transitions on scroll
- Fully responsive design for all devices

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.
